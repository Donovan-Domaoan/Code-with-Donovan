Cogniface

Built a full-stack face recognition app where users can register, sign in, and detect faces in images. Used React for the frontend, Node.js and Express for the backend, and PostgreSQL to handle user data. Everything’s fully connected and live — frontend, backend, database — all working together.

ChartBot

AI Stock Analyzer is a Python-powered app that uses ChatGPT and LangChain to break down stock trends and company info in plain English. Just type in a stock, and it gives you smart, AI-generated insights—no finance degree required.

Dirt to Dollars

Developed a regression model to estimate used bulldozer prices from historical auction data. Used algorithms like Random Forest Regression with feature engineering and cross-validation. Aimed at improving price transparency for heavy equipment sales.

Project link
TechPulse

TechPulse is a software engineering project that scrapes and filters top-rated stories from a trusted technology and cybersecurity news outlet. By leveraging web scraping techniques, the project provides curated, up-to-date content, ensuring that users stay informed about the latest trends and developments in the tech and cybersecurity fields.

Project link
CardioML

Built a machine learning model to predict heart disease using structured clinical data. Applied classification algorithms and evaluated performance using ROC AUC and precision-recall. Includes feature analysis and model explainability.
